# mood·me — 클릭 가능한 프로토타입

PRD(`docs/prd/mood-me-prd.md`)의 전체 여정을 백엔드·API 없이 체험하는 정적 프로토타입입니다.
로그인 → 메인 → 추구미 테스트 → 생성중 → 편집 → 결과·공유까지 실제로 클릭해 볼 수 있어요.

## 실행

의존성·빌드가 없습니다. 아무 방법으로나 여세요:

```bash
# 방법 1) 파일 직접 열기
open prototype/index.html

# 방법 2) 로컬 서버 (일부 브라우저의 폰트/이미지 정책상 권장)
python3 -m http.server 8777
# → http://localhost:8777/prototype/index.html
```

모바일 우선으로 설계됐습니다. 브라우저 개발자도구의 반응형 모드(예: 390×844)에서 보면 가장 자연스럽습니다.

## 프로토타입에서 목업(mock)으로 처리한 것

실제 서비스에서는 아래가 Next.js 라우트 + Supabase + Claude + fal.ai로 대체됩니다.

- **AI 생성**: 실제 이미지 생성 대신, 테스트에서 고른 사진들로 보드를 구성합니다. (이미지는 큐레이션된 Unsplash 사진)
- **무드 성향**: 답변에 매긴 축 가중치(`data.js`)로 레이더와 성향 이름을 계산합니다.
- **펜 도구·카카오 공유**: 실제 앱 연동 지점만 표시(안내 토스트).
- **PNG 내보내기**: 실제로 동작합니다 — 캔버스 좌표를 읽어 브라우저에서 이미지를 생성해 다운로드합니다.

## 디자인 방향

DESIGN.md의 북극성 **"채워지는 꿈, 밤에"** 를 따릅니다. 깊은 페트롤 잉크 배경 위에서 무드보드가
스스로 빛나는 광원처럼 떠오르고(결과물이 히어로), 따뜻한 애프리콧 글로우 하나로 친근함을 냅니다.
본문 Pretendard · 몽환의 정점 Gowun Batang, WCAG AA 대비·키보드·`prefers-reduced-motion` 대응.

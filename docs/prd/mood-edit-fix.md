지금부터 말하는 수정 요청은 무드 편집 페이지의 수정 요청이야 명심해.

1.  무드 편집 페이지의 헤더 저장 버튼 -> 저장 아이콘으로 변경, 버튼 사이즈도 icon으로 변경

2.  색상탭의 색상 선택 UI와 배경탭의 배경 선택 UI 통일
    배경탭의 버튼이 색상탭의 버튼 디자인으로 통일

3.  디자인시스템, button 컴포넌트에는 트렌스레이트 애니메이션이 없는데, 편집 페이지에 사용된 버튼들에 트렌스레이트 애니메이션이 사용됨 -> 애니메이션 삭제

4.  저장하기 버튼 기능과 다운로드 버튼 기능 나누기 헤더 우측에 다운로드 버튼 추가
    png로 저장, jpg로 저장은 다운로드 버튼을 누르면 등장
    다운로드 버튼 누르면 완성하고 공유하기 버튼과 계속 편집하기로 변경

5.  모달이 화면 중앙에 등장하도록 수정

6.  헤더 우측 화살표 버튼은 ghost 스타일로 변경

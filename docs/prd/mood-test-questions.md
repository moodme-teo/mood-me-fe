# 추구미 테스트 질문 (수집 → 확정)

> 🚧 **수집 중** (7/6~7/8) — **7/9 질문 확정 회의**에서 개수·내용·순서를 확정하고 이 문서를 갱신합니다.
> PRD [13. Open Questions #2](./mood-me-prd.md)를 해소하는 문서입니다. 확정 전까지 아래 질문 목록은 초안입니다.

## 수집 채널

- **이 문서의 PR 리뷰 코멘트**가 공식 수집 채널입니다 — 질문 아이디어·수정 의견을 해당 줄에 코멘트로 남겨주세요.
- 가벼운 아이디어는 Discord 채널에 던져도 됩니다 (7/9 회의 전에 PM이 이 문서로 취합).

## 설계 원칙 (PRD 근거)

- **선택지 타입은 3가지** (PRD 5.3): ① 이미지만 선택 ② 키워드만 선택 ③ 이미지 선택 + 텍스트 구체화
- **질문 개수는 8개 내외** — PRD 5.3 진척도 예시(`3 / 8`) 기준. 완료율이 핵심 지표(PRD 3)이므로 10개를 넘기지 않습니다.
- **모든 질문은 AI 프롬프트에 기여해야 합니다** (PRD 9): 선택 이미지 → 스타일 레퍼런스, 키워드/텍스트 → 텍스트 프롬프트. 프롬프트에 안 쓰이는 질문은 넣지 않습니다.
- **순서는 쉬운 것 → 구체화 순**: 직관적인 이미지 선택으로 시작해 몰입시키고, 타이핑이 필요한 구체화형은 마지막에 배치 (이탈 방지).
- 답변마다 상단 프리뷰가 "채워지는" 경험(PRD 5.3)과 연결되므로, 각 질문은 프리뷰에 반영될 시각 요소를 하나씩 더하는 구성이어야 합니다.

## 질문 초안 (8문항)

| # | 질문 | 타입 | 프롬프트 매핑 |
| --- | --- | --- | --- |
| 1 | 지금 가장 끌리는 분위기는? | ① 이미지 | 스타일 레퍼런스 (전체 무드 기준축) |
| 2 | 당신이 추구하는 무드를 골라주세요 (복수 선택) | ② 키워드 | 텍스트 프롬프트 — 무드 형용사 |
| 3 | 마음에 드는 색감은? | ① 이미지 | 컬러 팔레트 |
| 4 | 머물고 싶은 공간/장면은? | ① 이미지 | 배경·장면 요소 |
| 5 | 어울리고 싶은 계절과 시간대는? | ② 키워드 | 조명·톤 (예: 여름 해질녘, 겨울 아침) |
| 6 | 끌리는 질감/소재는? | ② 키워드 | 텍스처 (필름 그레인, 벨벳, 유리, 종이…) |
| 7 | 추구미에 가장 가까운 스타일 컷은? | ① 이미지 | 패션/피사체 스타일 레퍼런스 |
| 8 | 이 이미지를 고른 나, 한 마디로 표현하면? | ③ 이미지+텍스트 | 사용자 정체성 키워드 — 프롬프트 마무리 + 무드 프로파일 축 |

**타입 배분 초안**: 이미지형 4 · 키워드형 3 · 이미지+텍스트형 1 (마지막 1개만 — 타이핑 부담 최소화)

### 질문별 선택지 초안

- **Q2 무드 키워드**(복수 선택): 몽환적인 · 미니멀 · 빈티지 · 시크 · 러블리 · 스트릿 · 아늑한 · 청량한 · 그런지 · 클래식
- **Q5 계절·시간대**: 봄/여름/가을/겨울 × 새벽/한낮/해질녘/밤 (2단 선택 또는 조합 칩)
- **Q6 질감·소재**: 필름 그레인 · 벨벳 · 린넨 · 유리 · 메탈 · 종이 · 니트 · 물기
- 이미지형(Q1·3·4·7) 선택지 이미지는 `docs/work/todo/reference/`에 수집 — 문항당 4~6장, 스타일이 뚜렷이 갈리게

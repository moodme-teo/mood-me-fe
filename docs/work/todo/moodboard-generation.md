# 무드보드 생성 스펙 (B안) — 큐레이션 라이브러리 · 레이아웃 템플릿 · 생성 파이프라인

> 7/8 확정. 근거: 실사용 비전보드 레퍼런스 17장 분석 + 7/8 논의. 질문지 스펙은 [mood-test-questions.md](./mood-test-questions.md), 카드 이미지 수집 대장은 [mood-test-image.md](./mood-test-image.md).

## B안 핵심

**질문지 선택 카드 36장(`public/test-image/aesthetic/`)은 측정 도구일 뿐, 보드에 올라가지 않는다.** 보드의 큐레이션 타일은 해석 결과(persona_scores·keywords) 기반으로 별도 라이브러리에서 선정한다. 생성중 화면에서 카드들이 흩어졌다가 큐레이션 타일·AI 컷으로 재구성되는 서사로 전환한다.

## 레퍼런스에서 도출한 보드 문법 (17장 분석)

- **밀도**: 사진 타일 **최소 10 · 평균 15 · 최대 20장**이 겹치며 채워진다.
- **사진만으로는 보드가 안 된다** — 모든 레퍼런스가 4계층 조합: 사진 타일 + 문구 요소(손글씨 어퍼메이션·포스트잇·말풍선·티켓) + 배경 제거 오브제 컷아웃(클로버·딸기·나비·별 등) + 데코(테이프·찢은 가장자리·폴라로이드/금박 프레임·낙서 선).
- **중앙 타이틀**: 절반 이상이 "2026 VISION BOARD" 타이틀 카드를 중앙 히어로로 사용.
- **톤 통일은 편집 장치가 만든다** — 사진 개별 질감보다 배경 텍스처·프레임·팔레트가 보드의 결을 결정.
- **인물**: 뒷모습·손·신체 일부 컷이 자기 투영을 담당 (정면 얼굴은 서비스 기준상 배제 유지).

## 큐레이션 라이브러리 — 4계층

| 계층 | 규모 | 형태 | 위치 / 네이밍 |
| --- | --- | --- | --- |
| A. 사진 타일 | **80장** | jpg, 세로·가로·정방 혼합, 긴 변 800px+ | `public/test-image/board/` `b01~b80` |
| B. 오브제 컷아웃 | 30개 내외 | 투명 PNG (클로버·딸기·리본·나비·별·비행기 등) | `public/test-image/board-asset/object/` `o01~` |
| C. 문구 스티커 프레임 | 10종 내외 | 포스트잇·말풍선·티켓·테이프라벨 배경 에셋 — 문구 자체는 Claude 출력 + Konva Text 렌더 | `public/test-image/board-asset/sticker/` `s01~` |
| D. 데코 에셋 | 15~20개 | 테이프 조각·찢은 가장자리 마스크·폴라로이드/금박 프레임·낙서 SVG (톤 중립) | `public/test-image/board-asset/deco/` `d01~` |

- **태깅 (A)**: 카드 풀과 동일 구조 — 프롬프트 태그 + **페르소나 가중치 2~3개** + **무드 벡터 5축 값**. 전속 배정(20종×4장)이 아니라 다중 페르소나 공유로, 80장으로 페르소나당 유효 풀 8~10장을 확보한다 (80×2.5÷20). 무드 벡터 값은 조립 시 톤 일관성 필터의 열쇠.
- **태깅 (B)**: 페르소나/톤 태그만. C·D는 톤 프리셋 소속만 기록.
- **스타일 (A)**: 감성 디테일 컷 유지 + 카드 풀보다 **인물 뒷모습·손 컷 비중 상향** (자기 투영 담당). 얼굴 정면 금지.

## 레이아웃 템플릿 — 아키타입 6종 × 톤 프리셋 3~4종 ≈ 20 조합

타일 수가 10~20으로 변하므로 슬롯 고정 템플릿이 아니라 **배치 규칙(히어로 존 · 중형 존 · 필러 존 + 밀도 규칙)**으로 정의한다. Konva 조립 로직이 규칙 기반으로 채운다.

구도 아키타입 (레퍼런스 유형화):

1. **중앙 타이틀 방사형** — 타이틀 카드 중심, 타일이 둘러쌈 (가장 흔함)
2. **풀블리드 패치워크** — 여백 없이 가장자리까지 밀집, 타이틀 오버레이
3. **보드판** — 펠트/코르크 배경 + 프레임, 폴라로이드 흩뿌림
4. **세로 흐름형** — 위→아래 흐르며 곡선 텍스트 패스 관통 (폰 배경화면형)
5. **여백 스크랩북** — 배경 노출, 기울어진 폴라로이드 + 손글씨 주석·화살표
6. **매거진 그리드** — 정갈한 그리드 + 컬러칩

**톤 프리셋** = 배경 텍스처 + 프레임 스타일 + 데코 세트 묶음, 무드 벡터에서 도출 (3~4종).

## 생성 확정안

- **조립**: Konva 요소 배치(`MoodboardElement[]`)로 조립. 100% 생성 이미지는 배제.
- **구성비**: 사진 타일 10~20장 중 **AI 컷 30~50%**, 나머지 큐레이션 타일. **히어로 슬롯 = AI 컷.**
- **문구 스티커**: Claude가 뽑은 3개 중 **기본 2~4개 배치** (기존 "기본 1개"에서 상향 — 레퍼런스 밀도 근거).
- **모델**: **Gemini 2.5 Flash Image 기본** — fal 클라이언트 유지, 모델 id만 교체. **큐레이션 타일 2~3장을 레퍼런스로 넣어 톤 앵커링.**
- **질감 방침**: AI 프롬프트에 질감 지시를 넣지 않는다 (필름 그레인·빈티지 강제는 slop 지름길). 질감·통일감은 **조립 레이어**가 담당 — 타일별 프레임/가장자리 마스크, 톤 프리셋별 배경 텍스처, 선택적 저불투명 그레인 오버레이 1장 (채택 여부는 #41 구간 3에서 비교 판정).

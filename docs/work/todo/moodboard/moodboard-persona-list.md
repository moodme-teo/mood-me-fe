# 페르소나 사전 — 느낌 · 이미지 키워드

> 📌 [moodboard-creation.md](./moodboard-creation.md) §보드 이미지 생성 흐름 **1단계**의 본문. 분량 때문에 분리했다. **이 파일이 키워드의 정본이다.**
>
> 조립 규칙(비율 → 키워드 개수)은 본 문서 §3에 있다.

## 사전 (20종)

20종 = **미학 코어 14 (C)** + **인생 테마 6 (T)**. 코어는 비주얼, 테마는 태도. 3단계 프롬프트는 코어 키워드로 "룩"을, 테마 키워드로 "삶의 장면"을 잡는다.

키워드는 **배열**로 정의한다. 개수는 §3 조립 규칙이 비율에서 계산하며, **배열에서 균등 간격으로** 뽑는다.

- **코어 14종 = 각 20개 · 테마 6종 = 각 10개** (총 340개). 이는 §3의 예산(코어 20 · 테마 10)과 같은 수 — **한 페르소나가 100%를 먹어도 상한에 걸리지 않게** 맞춘 값이다.
- **사전 키는 `persona.ts`의 페르소나 이름을 접두사로 갖는다.** 9종은 `클린 걸 · 댓 걸`처럼 별칭 접미사가 붙어 이름과 정확히 일치하지 않는다 — 조립 코드는 `키 === 이름 || 키.startsWith(이름)` 으로 찾아야 한다. (`맥시멀 글램`은 2026-07-09에 `맥시멀리스트 글램`에서 통일했다. 접두사 매칭조차 안 되던 유일한 키였다.)
- 배열 순서는 **`작은 오브제 → 의상·질감 → 큰 장면`**. 지배 페르소나는 균등 간격 덕에 소품부터 큰 장면까지 고루 받고, **꼬리(1–9%)는 앞의 1–2개(작은 오브제)만 받는다.**
